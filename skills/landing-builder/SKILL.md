# Landing Page Builder

Взима референтен URL (дизайн/стил) + продуктов URL (съдържание) и генерира готова HTML/CSS лендинг страница.

## Употреба
1. Подай **референтен URL** — страницата чийто дизайн искаш да копираш
2. Подай **продуктов URL** — страницата от която да извлече съдържание
3. Skill-ът скрейпва и двете, комбинира ги и генерира `landing.html`

## Input
- `source_url` — URL на референтната страница (дизайн)
- `product_url` — URL на продуктовата страница (съдържание)
- `language` — език на генерираната страница (по подразбиране: български)

## Output
- `landing.html` — готова лендинг страница с CSS inline, responsive, готова за публикуване
