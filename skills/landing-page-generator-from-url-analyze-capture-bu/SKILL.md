# # Landing Page Generator from URL — Analyze, Capture & Build

Generates a professional, high-converting landing page by extracting design inspiration and content from any given URL, using multiple frontend frameworks and strong UX/UI principles.

## Description
This skill captures a full-length screenshot of the target webpage and extracts all key visual assets, typography, color schemes, and textual content. It then uses that analysis to design and code a brand-new, responsive landing page with modern frameworks like Tailwind CSS, GSAP, or Alpine.js—elevating the original design with better UX patterns, animations, and conversion-focused layouts. The final output is a ready-to-deploy page that mirrors the source’s intent while adding professional polish.

## When to Use
- When you need to quickly create a stunning landing page inspired by an existing website’s structure and content.
- When you want to rebuild or modernize a competitor’s landing page with improved UX, interactivity, and responsiveness.
- When a client asks for a professional, multi-framework landing page based on a reference URL but with enhanced visual appeal and better conversion flow.

## Input
- A target URL from which a full-page screenshot will be taken and content will be scraped.
- Optional preferences: desired frontend frameworks (e.g., Tailwind, Bootstrap, GSAP), tone, color palette overrides, specific sections to include (hero, features, testimonials, CTA).

## Output
- A complete, responsive HTML/CSS/JS landing page (single file or organized project) using the chosen frameworks and UI patterns.
- A full-page screenshot of the original URL for visual reference.
- A summary of extracted design tokens, content structure, and decisions made during the redesign.

## Primary Agent
**opencode** — Ideal for orchestrating headless browser screenshots, parsing HTML/CSS, and generating complex multi-framework landing pages. It handles file operations, code generation, and integration of various libraries seamlessly.

## Example Prompts
- "Build a landing page from https://linear.app, capture the screenshot, extract its minimal design, and recreate it with Tailwind CSS, GSAP animations, and a modern bento-grid layout."
- "Take https://www.apple.com/macbook-air as inspiration, screenshot the page, extract the hero and feature sections, then generate a new landing page using Bootstrap 5 and Alpine.js with an improved mobile UX."
